⚙️Construcción

⚒️ Pasos

Materiales necesarios

  • Sensor de colores (TCS34725 o TCS3200)
  • Brazo robótico con servomotores
  • Placa Arduino MEGA
  • Cables jumper macho-hembra o macho-macho
  • Protoboard 
  • Fuente de alimentación o batería
  • Computadora con mBlock o Arduino IDE

⚒️ Pasos para la incorporación

  1. Ubica el sensor de colores en el brazo robótico, preferiblemente en una posición donde pueda "ver" fácilmente el objeto a analizar, como cerca de la pinza o en un brazo lateral.
  2. Conecta los cables del sensor a la placa siguiendo este esquema básico:

VCC del sensor → 5V o 3.3V en la placa

GND del sensor → GND en la placa

SDA (línea de datos) → Pin SDA en Arduino (generalmente A4)

SCL (línea de reloj) → Pin SCL en Arduino (generalmente A5)

(Para sensores digitales tipo TCS3200, se conectan pines de salida a pines digitales, consulta datasheet específico)

3. Asegura las conexiones con cuidado para evitar falsos contactos o desconexiones durante el movimiento del brazo.

4. Fija el sensor al brazo con cinta adhesiva, tornillos pequeños o una base impresa en 3D, asegurándote que quede estable y no se mueva.

5. Verifica la alimentación eléctrica para que el sensor y el brazo funcionen sin interrupciones. Si usas batería, confirma que está cargada.

6. Prueba la comunicación entre el sensor y la placa cargando un programa de lectura básica, para confirmar que el sensor detecta colores correctamente.

7. Integra el sensor con la programación del brazo, para que el robot pueda reaccionar según el color detectado.
