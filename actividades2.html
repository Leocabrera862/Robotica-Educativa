🧩 Actividad #1

🏗️ Apila bloques con el brazo robótico

🎯 ¿Qué vamos a hacer?
En esta actividad, programaremos el brazo robótico para que levante pequeños bloques y los apile uno sobre otro, como si estuviera construyendo una torre.

🧠 ¿Qué aprenderás?

  • Controlar el movimiento del brazo con precisión.
  • Planificar una secuencia de acciones paso a paso.
  • Programar movimientos usando bloques en mBlock.
  • Desarrollar paciencia, lógica y coordinación.

🛠️ Materiales necesarios:

  • Brazo robótico funcional
  • 3 o más cubos pequeños (pueden ser bloques de LEGO, cajas pequeñas, tapitas)
  • Zona de trabajo plana
  • Placa Arduino MEGA
  • Computadora con mBlock
  • Fuente de energía o batería

📋 Pasos a seguir:

  1. Prepara el área de trabajo:
  2. Coloca los bloques en fila (no apilados), dejando espacio entre ellos y una zona designada para construir la torre.
  3. Programa el primer movimiento:
  4. Haz que el brazo se desplace hacia el primer bloque, lo agarre con la pinza y lo lleve al centro de construcción.
  5. Apila uno a uno:
  6. Repite el proceso con los otros bloques. Cada vez el brazo debe colocar el siguiente encima del anterior, ajustando la altura.
  7. Controla la precisión:
  8. Ajusta la programación para que el brazo no derribe la torre. Puedes probar varias veces hasta que logres una torre estable.
    Agrega un reto extra (opcional):
  9. Usa bloques de diferentes tamaños.
  10. Crea una torre siguiendo un patrón de colores.
  11. ¡Haz una competencia por equipos!
